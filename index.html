บทนำ

ในบทเรียนนี้ คุณจะได้เรียนรู้วิธีการใช้ Python สำหรับการคำนวณทางคณิตศาสตร์

การคำนวณทางคณิตศาสตร์พื้นฐาน

เราจะเริ่มต้นด้วยการคำนวณพื้นฐานใน Python เช่น การบวก, การลบ, การคูณ และการหาร

ตัวอย่าง: การบวกใน Python

                
# การบวก
a = 10
b = 5
result = a + b
print(result)  # ผลลัพธ์: 15

ในตัวอย่างข้างต้น เราได้ทำการบวกสองตัวเลข

การใช้ฟังก์ชันทางคณิตศาสตร์ใน Python

Python มีห้องสมุด math ที่ให้เราสามารถทำการคำนวณที่ซับซ้อนได้มากขึ้น

ตัวอย่าง: การหาค่ารากที่สอง

                
import math
number = 25
result = math.sqrt(number)
print(result)  # ผลลัพธ์: 5.0

ตัวอย่างนี้แสดงวิธีการคำนวณค่ารากที่สองโดยใช้ฟังก์ชัน sqrt จากห้องสมุด math
